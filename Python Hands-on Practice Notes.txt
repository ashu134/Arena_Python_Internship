2 Create Variables to Store Different Types of Data

Variables store information that the program uses.

name = "Ashu"        # string
after_years = 25      # integer
height = 5.8          # float
is_student = True     # boolean


To display values:

print(name, after_years, height, is_student)


3 Build a Simple Calculator for Basic Math Operations

a = int(input("Enter first number: "))
b = int(input("Enter second number: "))

print("Addition:", a + b)
print("Subtraction:", a - b)
print("Multiplication:", a * b)
print("Division:", a / b)

`int()` converts user input from string to integer.
